RiotKit Backups initiative

A safe place for organization's data.

We are an anarchist technical organization

The goal is to provide data-safety mechanisms for each anarchist organization.
Privacy, automation and high-level security is very important.

Anarchist movement deserves modern solutions, let's take over the technical knowledge from capitalists and bring it into the grassroot organizations.

Cheap

More organizations sharing same space, less cost of the server

Managed

You don't have to care about many things, just send encrypted data, we store it as is without having a possibility to know what you store

Secure

Each organization is expected to encrypt backups with own keys (we provide a tool for it) before sending to us. Nobody can read this data except the organization that has a key.

Modern

Simply setup once, and don't care much. It's automatic. On any failure you can restore a one of the copy from the past.

Transparent

All we do we publish as open source. If you don't trust us for any reason, just create your own infrastructure from our public templates. [1] [2] [3]

Distributed

If you are undecided, because all data may be in our hands regardless if we have access to it or not. In case our server would disappear what would happen? Our server is mirrored. We have multiple servers that keeps the same data in different locations. You can also have a read-only replica if you would like to host it.

Use the infrastructure

If organization you are in is anarchist-rules based, then we can contact to join the shared backups storage.

Develop the infrastructure

Development and administration - if you are a technical person who knows about the infrastructure or about the code, then you can help us via github, or by contacting us.
